Con el plugin SEO Links podrás añadir de forma automática enlaces estratégicos a palabras clave deseadas en las entradas de tu blog de Wordpress.
 
== Descripción ==
 
SEO Links es un plugin para Wordpress con el que podrás añadir de forma automática enlaces estratégicos a palabras clave deseadas en las entradas de tu blog. El plugin efectuará una búsqueda en tu entrada con la palabra clave que elijas y le asociará el enlace que hayas determinado previamente. Si la palabra clave se repite en varias ocasiones, solo se enlazará el primer resultado. 
 
Con SEO Links podrás añadir fácilmente un title, decidir si el enlace es “follow” o “no follow” o si quieres que se abra en “blank”, “parent”, “self” o “top”. Con este plugin además tienes acceso a un listado desde donde puedes gestionar los enlaces ya creados, pudiendo consultar fácilmente cuántas veces ha sido abierto cada uno.  

== Instalación ==
 
FROM YOUR WORDPRESS DASHBOARD

Visita ‘Plugins > Añadir nuevo’
Buscar ‘Ninja SEO Links’
Activa Ninja SEO Links desde tu página de Plugins.

Descargar Ninja SEO Links.
Sube el directorio Ninja SEO Links a tu directorio ‘/wp-content/plugins/’, usando tu método favorito (ftp, sftp, scp, etc…)
Activa Ninja SEO Links desde tu página de Plugins.
