With Ninja SEO Links plugin you can automatically add strategic links to desired keywords in the entries of your Wordpress blog.
 
== Description ==
 
Ninja SEO Links is a Wordpress plugin with which you can automatically add strategic links to your desired keywords in your blog posts. The plugin will perform a search in your entry with the keyword of your choice and will associate the link you have previously determined. If the keyword is repeated, only the first result will be linked.

With Ninja SEO Links you can easily add a title, decide if the link is "follow" or "no follow" or if you want it to open in "blank", "parent", "self" or "top". With this plugin you also have access to a list from where you can manage the links already created, you can easily consult how many times it has been opened each one.

== Installation ==
 
FROM YOUR WORDPRESS DASHBOARD

Visit ‘Plugins > Add New’
Search for ‘Ninja SEO Links’
Activate Ninja SEO Links from your Plugins page.

Download Ninja SEO Links.
Upload the Ninja SEO Links directory to your ‘/wp-content/plugins/’ directory, using your favorite method (ftp, sftp, scp, etc…)
Activate Ninja SEO Links from your Plugins page.
